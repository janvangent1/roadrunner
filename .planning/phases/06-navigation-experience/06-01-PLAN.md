  <behavior>
    OffRouteDetector.minDistanceToPolyline tests:
    - Point ON the polyline → returns 0.0 (within floating-point tolerance)
    - Point 100m perpendicular to a horizontal segment → returns ~100.0 (within 2m tolerance)
    - Point past the end of a segment → returns distance to nearest endpoint, not the perpendicular projection
    - Empty polyline list → returns Double.MAX_VALUE (safe sentinel, never triggers off-route)
    - Single-point polyline → returns haversine distance from point to that single point
  </behavior>
  <action>
    Create OffRouteDetector.kt in package `com.roadrunner.app.navigation`:
    - Object (no DI needed — pure math utility)
    - `fun minDistanceToPolyline(point: GeoPoint, polyline: List<GeoPoint>): Double`
      - Returns `Double.MAX_VALUE` if polyline is empty
      - For single-point list, returns haversineMetres(point, polyline[0])
      - For 2+ points: `polyline.zipWithNext().minOf { (a, b) -> distanceToSegment(point, a, b) }`
    - Private `fun distanceToSegment(p: GeoPoint, a: GeoPoint, b: GeoPoint): Double`:
      - Project p onto segment ab using dot-product parametric formula in lat/lon space (approximation valid for short segments)
      - Clamp t to [0.0, 1.0]; compute nearest point; return haversineMetres(p, nearest)
    - Private `fun haversineMetres(a: GeoPoint, b: GeoPoint): Double`: standard haversine with Earth radius 6371000.0

    Create NavigationViewModel.kt in package `com.roadrunner.app.ui.navigation`:
    - Annotate `@HiltViewModel`, inject `LocationRepository`, `RouteRepository`, `SavedStateHandle`
    - `val routeId: String = checkNotNull(savedStateHandle["routeId"])`
    - Expose a single `val uiState: StateFlow<NavigationUiState>` backed by `MutableStateFlow`
    - `data class NavigationUiState(val location: Location? = null, val speedKmh: Float = 0f, val distanceCoveredKm: Double = 0.0, val distanceRemainingKm: Double = 0.0, val elapsedSeconds: Long = 0L, val isOffRoute: Boolean = false, val routePoints: List<GeoPoint> = emptyList(), val waypoints: List<WaypointDto> = emptyList(), val isLoading: Boolean = true, val error: String? = null)`
    - `init` block: launch `loadRoute()` which calls `routeRepository.getRoute(routeId)`, parses GPX via `routeRepository.getDecryptedGpx(routeId)` using `io.ticofab.androidgpxparser.parser.GPXParser` to extract track points as `List<GeoPoint>`, stores `totalDistanceKm` from route, stores waypoints from route.
    - GPX parsing: `GPXParser().parse(ByteArrayInputStream(gpxBytes))` returns a `Gpx` object; extract `gpx.tracks[0].trackSegments[0].trackPoints` → map each to `GeoPoint(trackPoint.latitude, trackPoint.longitude)`.
    - `fun startTracking()`: calls `locationRepository.startLocationUpdates()`, then in `viewModelScope.launch` collect `locationRepository.locationFlow`. On each non-null location:
      - Update `speedKmh = location.speed * 3.6f` (m/s to km/h)
      - Accumulate `distanceCoveredKm` using haversine between previous and current position
      - `distanceRemainingKm = (totalDistanceKm - distanceCoveredKm).coerceAtLeast(0.0)`
      - Check off-route: `val dist = OffRouteDetector.minDistanceToPolyline(GeoPoint(lat, lng), routePoints)`. Apply hysteresis: set `isOffRoute = true` if dist > 50.0 and currently on-route; set `isOffRoute = false` if dist < 40.0 and currently off-route.
    - `fun stopTracking()`: calls `locationRepository.stopLocationUpdates()`
    - Start an elapsed-time ticker in `viewModelScope`: `while(true) { delay(1000L); _uiState.update { it.copy(elapsedSeconds = it.elapsedSeconds + 1) } }` — start only after route loaded successfully.
    - `override fun onCleared()`: call `stopTracking()`.
