**backend/src/middleware/requireAuth.ts** — export `requireAuth` as a Fastify `preHandler` hook. Call `await request.jwtVerify()` to verify the Bearer token from the Authorization header. On failure, reply 401 with `{ error: 'Unauthorized' }`.

**backend/src/middleware/requireAdmin.ts** — export `requireAdmin` as a `preHandler` hook. Must be used AFTER `requireAuth`. Checks `request.user.role === 'ADMIN'`. If not, reply 403 with `{ error: 'Forbidden' }`.

**backend/src/routes/auth.ts** — export `authRoutes` as a Fastify plugin (use `fastify-plugin` to avoid scope encapsulation). Implement five routes with Zod validation on request bodies:

- `POST /register` — body: `{ email: string, password: string (min 8) }`. Hash password with bcrypt (saltRounds=12). Create User with `role: 'USER'`. Return `{ accessToken, refreshToken, user: { id, email, role } }`. Return 409 if email already exists.

- `POST /login` — body: `{ email, password }`. Find user by email (throw 401 if not found). Compare bcrypt hash. Generate access JWT (`{ sub: userId, role }`, expires 15m) via `fastify.jwt.sign()`. Generate refresh token (UUID v4), store hashed with bcrypt in `refresh_tokens` table (expires 7 days). Return `{ accessToken, refreshToken, user: { id, email, role } }`.

- `POST /google` — body: `{ idToken: string }`. Verify the Google ID token using `google-auth-library` `OAuth2Client.verifyIdToken({ idToken, audience: process.env.GOOGLE_CLIENT_ID })`. Extract `email`, `sub` (Google sub). Upsert user: find by `googleId` or email; if found by email, update `googleId`; if not found, create with `role: 'USER'`. Return same token shape as /login.

- `POST /refresh` — body: `{ refreshToken: string }`. Find the refresh token record (not expired, not revoked). Verify the stored bcrypt hash. Revoke old token (set `revokedAt = now()`). Create new refresh token. Return `{ accessToken, refreshToken }`.

- `POST /logout` — preHandler: `requireAuth`. Body: `{ refreshToken: string }`. Find and revoke the token by setting `revokedAt = now()`. Return 204.

**backend/src/app.ts** — add `app.register(authRoutes, { prefix: '/api/v1/auth' })` import and registration. Import `prisma` from `lib/prisma` and `redis` from `lib/redis` and add a graceful shutdown hook to disconnect both on SIGTERM.

Install additional dependency: `bcrypt` + `@types/bcrypt`, `fastify-plugin`, `uuid` + `@types/uuid`.
