```kotlin
@Singleton
class LicenseRepository @Inject constructor(
    private val apiService: ApiService,
) {
    private val _cachedLicenses = mutableListOf<LicenseDto>()

    /** Fetch all user licenses from server; update in-memory cache. */
    suspend fun getMyLicenses(): Result<List<LicenseDto>> { ... }

    /**
     * Compute display LicenseStatus for a given routeId from the cached list.
     * Returns Triple(LicenseStatus, expiresAt, LicenseType?).
     * Uses server-issued expiresAt — does NOT call System.currentTimeMillis() for active/expired decision.
     * (System.currentTimeMillis() is used only for the EXPIRING_SOON threshold, which is a display hint
     * not a security gate — device clock tampering here only affects the warning banner, not access.)
     */
    fun computeLicenseStatus(routeId: String): Triple<LicenseStatus, String?, LicenseType?> { ... }

    /**
     * Call POST /licenses/check for a specific route before navigation start.
     * Returns Result<NavigationSession> on success (valid license).
     * Returns Result.failure with a user-readable message on 403/error:
     *   "License expired" for reason EXPIRED
     *   "No license found" for reason NOT_FOUND
     *   "License revoked" for reason REVOKED
     *   "License check failed" for network errors
     */
    suspend fun checkLicense(routeId: String): Result<NavigationSession> { ... }
}
```

Implementation details:

**getMyLicenses():**
- Call `apiService.getMyLicenses()`
- On success: clear `_cachedLicenses`, addAll from response body, return Result.success(list)
- On failure/non-2xx: return Result.failure — do NOT clear cache on failure (keep stale data)

**computeLicenseStatus(routeId):**
- Find license in `_cachedLicenses` where `routeId` matches and `revokedAt == null`
- If not found: return Triple(LicenseStatus.AVAILABLE, null, null)
- If found with `type == LicenseType.PERMANENT` or `expiresAt == null`: return Triple(LicenseStatus.OWNED, null, LicenseType.PERMANENT)
- If found with `expiresAt` present:
  - Parse with `Instant.parse(expiresAt)`
  - `val nowMillis = System.currentTimeMillis()` — acceptable here (display hint only, not security gate)
  - `val expiryMillis = instant.toEpochMilli()`
  - if `expiryMillis < nowMillis`: return Triple(EXPIRED, expiresAt, type)
  - if `(expiryMillis - nowMillis) <= 3 * 24 * 60 * 60 * 1000L`: return Triple(EXPIRING_SOON, expiresAt, type)
  - else: return Triple(ACTIVE, expiresAt, type)

**checkLicense(routeId):**
- Call `apiService.checkLicense(LicenseCheckRequest(routeId))`
- On 200 success: map body to `NavigationSession(sessionToken = body.sessionToken, sessionExpiresAt = body.expiresAt ?: ..., licenseType = body.licenseType)`
  - Note: `LicenseCheckResponse.expiresAt` is `licenseExpiresAt` in the server response; the server also returns `sessionExpiresAt` separately. The current `LicenseCheckResponse` DTO only maps `expiresAt` — use that field as the session expiry for NavigationSession (it represents when the session window closes, 1 hour from server).
  - Actually re-read `LicenseCheckResponse`: `{ sessionToken, expiresAt, licenseType }` — in context this `expiresAt` is `licenseExpiresAt` from the backend. The session window expiry (`sessionExpiresAt`) is not yet in the DTO. Add `sessionExpiresAt: String?` to `LicenseCheckResponse` in `RouteDtos.kt` and use that for `NavigationSession.sessionExpiresAt`. Fall back to `expiresAt` if `sessionExpiresAt` is null.
- On 403 response: parse error body JSON `{ code: "EXPIRED"|"NOT_FOUND"|"REVOKED" }` — map to user-readable string; return Result.failure(Exception("License expired" / "No license found" / "License revoked"))
- On network exception: return Result.failure(Exception("License check failed"))

Also update `RouteDtos.kt` to add `sessionExpiresAt: String? = null` to `LicenseCheckResponse` so the DTO captures the server-issued session window expiry.

Imports needed: `java.time.Instant`, `javax.inject.Inject`, `javax.inject.Singleton`, `dagger.hilt.android.lifecycle.HiltViewModel` (not needed here), all dto types.
